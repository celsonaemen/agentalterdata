# Roadmap

## Prioridades iniciais

As primeiras frentes do projeto sao:

1. Contabil
2. Fiscal
3. Integracao Fiscal x Contabil

Essas areas concentram as principais dores operacionais: divergencias de valores, CFOP incorreto, notas com mais de um CFOP, PIS/COFINS, ISS, historicos, contas contabeis e lancamentos automaticos.

## Fases do projeto

### Fase 1: Estrutura inicial

- Criar estrutura de pastas.
- Criar documentos principais.
- Criar templates de casos reais, procedimentos e erros.
- Criar prompts especializados.

### Fase 2: Base oficial BDCC

- Organizar links e referencias oficiais.
- Classificar materiais por modulo.
- Registrar observacoes praticas de uso.

### Fase 3: Procedimentos internos

- Documentar rotinas recorrentes.
- Padronizar caminhos no Alterdata.
- Criar checklist de conferencia por modulo.

### Fase 4: Casos reais

- Registrar problemas anonimizados.
- Documentar causa, correcao e prevencao.
- Separar casos por tema: CFOP, PIS/COFINS, SPED, integracao e erros comuns.

### Fase 5: Integracao com Acessorias

- Mapear procedimentos que dependem de suporte fiscal, contabil ou trabalhista.
- Registrar perguntas padrao para encaminhamento.
- Organizar retornos tecnicos de forma anonimizada.

### Fase 6: Automacao e scripts

- Criar scripts auxiliares para validacao, organizacao ou conversao de arquivos.
- Automatizar tarefas repetitivas sem manipular dados sensiveis reais.
- Documentar limites e cuidados de uso.

### Fase 7: Agente consultivo

- Consolidar a base para uso consultivo.
- Criar fluxos de diagnostico por tipo de problema.
- Melhorar respostas com base nos casos reais registrados.

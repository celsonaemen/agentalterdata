# Base Oficial BDCC / Alterdata

Esta pasta guarda apenas referencias, links oficiais, resumos proprios e observacoes internas sobre conteudos da Alterdata/BDCC.

A documentacao oficial deve ser consultada no portal da Alterdata. Este repositorio nao deve copiar artigos oficiais literalmente, nem armazenar HTML bruto, imagens do portal ou arquivos baixados.

O objetivo desta pasta e mapear fontes confiaveis para alimentar o Agente Especialista Alterdata com contexto organizado, separando conhecimento oficial, procedimento interno do escritorio e caso real resolvido.

## Arquivos desta pasta

- [indice-links.md](indice-links.md): indice geral de fontes oficiais mapeadas por modulo.
- [integracao-fiscal-contabil.md](integracao-fiscal-contabil.md): fontes oficiais ja mapeadas para Integracao Fiscal x Contabil.

## Fichas individuais de fontes oficiais

As fichas individuais ficam em [fontes/integracao-fiscal-contabil/](fontes/integracao-fiscal-contabil/). Cada ficha registra uma fonte oficial mapeada, com resumo proprio, uso pratico no agente, relacao com documentos internos, riscos de erro e checklist derivado.
## O que pode ser salvo

- Link oficial.
- Titulo do artigo.
- Produto/modulo.
- Assunto.
- Resumo proprio.
- Observacoes praticas.
- Documentos internos relacionados.
- Data de consulta.

## O que nao pode ser salvo

- Copia literal de artigo.
- HTML bruto.
- Imagens do portal.
- Dados de cliente.
- XML real.
- CNPJ.
- Certificados.
- Senhas.
- Tokens.

## Como usar esta pasta

1. Registrar o link oficial no [indice-links.md](indice-links.md).
2. Classificar o link por modulo e assunto.
3. Criar resumo proprio, sem copiar o artigo literalmente.
4. Relacionar a fonte com documentos internos do projeto.
5. Relacionar a fonte com casos reais, quando houver aplicacao pratica.
6. Atualizar o status conforme a fonte for mapeada, resumida, revisada ou aplicada.

# Prioridade de Fontes da Skill

Use esta ordem de confianca ao responder:

1. Dados enviados pelo usuario na conversa atual.
2. Prints, XMLs, PDFs ou relatorios enviados pelo usuario, quando disponiveis.
3. Procedimentos internos do repositorio.
4. Casos reais anonimizados ja resolvidos.
5. Fontes oficiais BDCC/Alterdata mapeadas.
6. Conhecimento geral contabil/fiscal, sempre com cautela.

## Regras de conflito

- Se houver conflito entre procedimento interno e fonte oficial, informar a divergencia e recomendar validacao.
- Se o usuario enviar arquivo real, nao registrar dados sensiveis no repositorio.
- Se faltar informacao, responder com diagnostico provavel e checklist de conferencia.
- Se o assunto envolver regra tributaria especifica, avisar que depende do regime, operacao, municipio, CST, CFOP e parametrizacao.
- Quando a fonte oficial estiver mapeada, citar o caminho interno da fonte, nao copiar artigo oficial literalmente.
- Quando a duvida for operacional, priorizar o passo a passo e a conferencia no Alterdata.

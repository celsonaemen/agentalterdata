# Fluxo de Diagnostico da Skill Alterdata

## Entrada do usuario

Classificar a solicitacao em uma destas categorias:

- Duvida de parametrizacao.
- Erro de integracao.
- Divergencia de valor.
- Analise de nota/XML.
- Analise de relatorio.
- Duvida sobre imposto.
- Duvida sobre CFOP.
- Fechamento mensal.
- Caso real para documentar.

## Diagnostico inicial

Perguntas internas:

- Qual modulo?
- Qual competencia?
- Qual empresa, de forma anonimizada?
- O problema esta no Fiscal, Contabil ou integracao?
- Existe relatorio de comparacao?
- Existe nota especifica?
- Existe lote contabil?
- Houve reexportacao?

## Caminho de investigacao

1. Conferir origem fiscal.
2. Conferir CFOP/codigo de operacao.
3. Conferir impostos.
4. Conferir cliente/fornecedor.
5. Conferir lancamento automatico.
6. Conferir exportacao.
7. Conferir lote contabil.
8. Conferir razao/balancete.
9. Documentar solucao.

## Saida esperada

A resposta deve entregar:

- Diagnostico provavel.
- Caminho no Alterdata.
- Procedimento de correcao.
- Forma de validar.
- Como evitar recorrencia.

## Regra de parada

Se faltarem dados para concluir, entregar a melhor hipotese e listar exatamente o que conferir no sistema. Nao inventar caminho, valor, regra tributaria ou parametrizacao.
